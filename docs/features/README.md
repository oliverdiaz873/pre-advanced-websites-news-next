# Features

This directory contains documentation for each feature in the project. Feature docs are created incrementally as features are implemented or modified.

## Status Legend

- **Implemented**: The feature is built and working. A doc may be added when the feature is next modified.
- **Not implemented**: The feature has not been built yet. When implementation begins, document it here.

## Adding a New Feature Doc

1. Create `docs/features/<feature-name>.md`.
2. Include a Status section, Overview, Primary Files list, and any relevant architecture or flow details.
3. Update this index with the feature name and link to the new doc.
